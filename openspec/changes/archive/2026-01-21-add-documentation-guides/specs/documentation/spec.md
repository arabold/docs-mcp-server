## ADDED Requirements
### Requirement: Documentation Guides
The system MUST provide comprehensive user documentation for configuration and client integration.

#### Scenario: Embedding Configuration Guide
- **WHEN** a user wants to configure a specific embedding provider (e.g., Ollama, Azure)
- **THEN** they can find a dedicated guide with exact configuration examples and environment variables

#### Scenario: Client Integration Guide
- **WHEN** a user wants to connect the server to an MCP client (e.g., Claude, Cline, Zed)
- **THEN** they can find a dedicated guide with specific configuration snippets for that client
